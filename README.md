# jodoro

## building locally
Create the `out` directory

```
mkdir out && cd out
```

here you'll put the `.class` files and crate the JAR file

```
javac -d . ../src/*.java
jar cfm jodoro.jar ../META-INF/MANIFEST.mf *.class ../media/*
java -jar jodoro.jar
```
